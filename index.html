Philosophy

デジタルな枯淡

江戸の美意識「わび・さび」は、不完全なもの、移ろいゆくものに美を見出す心です。
Edo-AI Bonsaiは、その精神をデジタルの世界で再解釈しました。

クラウド上の演算によって生まれる盆栽は、二度と同じ姿にはなりません。
それはまるで、自然界の木々が風雪に耐え、独自の枝ぶりを形成するように、
乱数という名の「デジタルの風」によって育てられます。

永遠に保存可能なデータではなく、
一期一会の生成結果にこそ、現代の「あはれ」があるのです。

The Algorithm

数式による成長

この盆栽は、再帰的なフラクタルアルゴリズムによって描かれています。
幹から枝へ、枝から葉へ。
単純なルールの繰り返しが、複雑で有機的な造形を生み出します。

function grow(branch) {
  if (energy < 0) return;
  split(branch);
  grow(left);
  grow(right);
}

葉の一枚一枚は、0と1のビット、あるいはピクセルの集合体。
遠目には自然な緑に見えても、近づけばそこには
デジタルな構成要素（グリッチ）が隠されています。
